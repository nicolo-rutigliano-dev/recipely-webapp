# Ricettario Online: Recipely WebApp



Un'applicazione web completa sviluppata per permettere agli utenti di creare, visualizzare, modificare e gestire la propria collezione di ricette personali. Questo progetto dimostra competenze **Full-Stack** nello sviluppo di applicazioni dinamiche e autenticate.

---

## Stack Tecnologico (MERN)

Il progetto è interamente costruito sullo **Stack MERN** (MongoDB, Express, React, Node.js), un set di tecnologie moderno e richiestissimo dal mercato.

| Componente | Tecnologia | Ruolo e Competenze Dimostrate |
| :--- | :--- | :--- |
| **Frontend** | **React.js** | Interfaccia utente dinamica, gestione dello stato tramite Hooks, e routing avanzato con React Router. |
| **Backend / API** | **Node.js** & **Express** | Creazione di **API RESTful** performanti per la gestione dei dati (logica di business). |
| **Database** | **MongoDB (NoSQL)** | Progettazione e interazione con un database orientato ai documenti tramite **Mongoose ODM**. |
| **Sicurezza** | **JSON Web Tokens (JWT)** | Implementazione di un sistema di autenticazione e autorizzazione stateless per proteggere le rotte sensibili. |
| **Deployment** | **Vercel / Render** | Gestione del deployment continuo (CI/CD) per rendere l'applicazione accessibile pubblicamente. |

---

## Funzionalità Chiave

* **Autenticazione Utente Completa:** Funzionalità di **Registrazione** e **Login** con hash delle password (Bcrypt) per la sicurezza.
* **CRUD Ricette:** Gli utenti autenticati possono **C**reare nuove ricette, **R**ileggere, **U**pdate e **D**elete solo quelle di loro proprietà.
* **Ricerca e Filtraggio:** Sistema di ricerca reattivo per trovare ricette per titolo, ingredienti o categoria.
* **Design Responsive:** L'interfaccia è stata ottimizzata con [Nome del framework CSS, es. Tailwind/Bootstrap] per l'uso su desktop, tablet e mobile.
